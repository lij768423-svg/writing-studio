# 02-outline: Anthropic 反超 OpenAI 真正引擎

## 写作角度

以 **Claude Code agentic coding 一年吃下 toB 开发者圈层**为切口，解释 Anthropic 在 3 个月内估值从 $852B 跌到 $965B（看错——是被反超到 $965B）的真正原因。

副线三条：Hassabis 公开警告 Mythos、IPO 三连发、OpenAI "missed revenue targets"。

不写成新闻通稿，**每节都要有反直觉判断**。

## 大纲（6 节，~3000 字）

### 一、5/28 那天发生了什么（~400 字）

开场用**一天里两件事**切入：
- 5/28 Anthropic 拿到 **$65B Series H**，估值 $965B
- 同日发 **Claude Opus 4.8**——但这个模型 Anthropic 自己说"比 Mythos 弱"

**反直觉点 1**：发一个"自承比自家兄弟弱"的模型，不是失误，**是策略**。Anthropic 在主动暴露自己还有更强的牌。

数据：
- 投资方：Altimeter / Dragoneer / Greenoaks / Sequoia / Coatue / ICONIQ
- Amazon 4 月追加 $25B（之前已有 $5B 投资）
- 估值跳升：3 个月前 $380B → 5 月 $965B（**2.5 倍**）

---

### 二、反超的真正引擎（~600 字）

**反直觉点 2**：估值反超的引擎**不是 Opus 4.8**（这只是官方放出来分散注意力的），**是 Claude Code**。

- 一年前 Claude Code 刚发布时定位是"实验性 agentic coding 助手"
- 一年后，Anthropic 整个 toB 收入跳到 $47B/年化，主要由 Claude Code 驱动
- "agentic coding" 重新定义了"程序员的工作"——从"逐行写代码"到"和 AI 结对编程"

加一段**个人体验**（用户是程序员 + 写代码的）：
- 我自己用 Claude Code 写代码确实改变了工作流（不能太长，3-5 句，呼应 5/30 那篇 AI 异化劳动的"过程异化"段）

判断：Claude Code 吃掉的是 toB 开发者圈层——这群人付费意愿高、LTV 高、不像 ChatGPT 那样"用了就走"。

---

### 三、收入曲线不会骗人（~500 字）

**反直觉点 3**：ARR 跳升的真正信号是**toB 收入比 toC 估值更真实**。

数字时间线：
- 2025 年底：$9-10B
- 2026 年初：$30B
- 2026-05：$47B

**5 个月翻 1.5 倍，一年翻近 5 倍**。这种曲线在 toB SaaS 行业几乎没见过（不是 ARR，是 annualized run rate）。

同期 OpenAI：
- 3 月融 $122B、估值 $852B
- 但 CNN 提到"missed revenue targets"
- 8 亿 ChatGPT 周活 vs 50 万 Claude 企业付费——**用户多不等于收入强**

判断：投资人终于发现"用户数 = 估值"逻辑不成立。**Anthropic 的"少用户、高 LTV"反而被市场重新定价**。

---

### 四、Hassabis 为什么在 I/O 上认输（~500 字）

**反直觉点 4**：Google CEO 在自家大会上**公开承认对手超越自己**。

5/28 Hassabis 在 Google I/O 2026 + Axios 采访：
- 罕见地没吹 Gemini
- 说 AGI 可能 2029–2030 到来
- 主动举例："Anthropic Mythos 让企业和政府措手不及"——"社会压力测试"

为什么？
- Mythos 是 Anthropic 5/28 同期公布但未详细说明的"顶级模型"——多家媒体提到它"和关键基础设施相关、已被 15+ 国家部署"
- Hassabis 用 Mythos 举例 = **等于 Google 高管公开承认 Anthropic 在某些维度领先**

LA Times 同期：「Google 内部把 AI 编程比赛送给 Anthropic 和 OpenAI」——这不是外部观察，是**内部人爆料**。

判断：Hassabis 的"AGI 警告"是给监管听的——但实际效果是**给 Google 解压"为什么 Gemini 落后"**。是个巧妙的政治动作。

---

### 五、IPO 三连发（~500 字）

**反直觉点 5**：AI 泡沫论 vs AI 公司扎堆上市，哪个对？

数据：
- Anthropic 6/1 秘密递交 S-1
- OpenAI 据报"未来几天/几周"递交
- SpaceX（已合并 xAI）5 月已递交，估值 $1.75T–$1.8T，融 $75B+

三件事**同一窗口**：
- 一方面：泡沫论说"AI 公司在烧钱"
- 另一方面：三家最有价值的公司同时段上市

判断：**上市 ≠ 泡沫破裂**。这三家上市是在 **套现 + 抢公开市场流动性**。Anthropic 选在估值 $965B 上市，而不是再等一年融 Series I，是**主动选择顶点套现**。

Wedbush Securities 分析师原话：「**IPO 市场 floodgates 打开了**」。

### 六、剩下的问题（~500 字）

收尾不做升华，回到"问号"。

三个未解之谜：
1. Mythos 到底是什么？媒体提到但描述模糊——可能与"国家级 AI"有关
2. OpenAI "missed revenue targets" 具体指什么？哪个季度？哪个区域？
3. FDE 模式（OpenAI 把工程师塞进客户公司）和 Palantir 模式是同一种吗？OpenAI 是不是在变成"咨询公司"？

最后留一个开放问题给读者：

> **当所有顶级 AI 公司都开始做"agent"、做"FDE"、做"垂直整合"——他们到底是在卖软件，还是在卖服务？**

如果这个边界模糊了，传统的"软件公司"估值模型可能要重写。这可能才是真正值得跟踪的信号。
