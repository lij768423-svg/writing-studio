### Part 0：理论框架 | 幻灯片 2-4 | ~3 分钟

**2. 三个我准备回答的问题**

- 经济：跨国 AI 资本如何循环？利润往哪流？
- 科技：算力、模型、数据的"国界"在哪里？
- 政治：美 / 中 / 欧三套 AI 规则如何博弈？

**3. 马克思会怎么看 AI**

- AI 不是"技术"，是**生产工具 + 生产资料 + 劳动对象**的三位一体
- 算力 ≈ 不动产（土地）；模型 ≈ 机器大工业；数据 ≈ 原材料
- AI 时代的"地租"：算力垄断溢价（NVIDIA 加速卡收入 3 年涨 1600%）

**4. 依附理论的提醒**

- 沃勒斯坦：中心生产高附加值产品，边缘生产原材料 + 承担污染
- AI 时代：中心（美/中）控制模型，边缘（全球南方）提供**数据 + 算力 + 矿**
- 谁在剥削谁？看利润流向 + 看规则制定权

### Part 1：经济层 | 幻灯片 5-10 | ~7 分钟

**5. AI 资本的金字塔**

```
塔尖：模型公司（Anthropic / OpenAI / Google）
       估值 $852B - $965B
塔身：基础设施（AWS / Azure / Google Cloud / xAI Colossus）
       2026 全球 capex $725B
塔基：硬件（NVIDIA / TSMC / 华为昇腾）
       NVIDIA 加速卡单季 $60.4B
底层：能源 + 矿
       NextEra 收 Dominion $67B（史上最大公用事业并购）
```

**6. 一笔算力账：Anthropic 付 xAI $1.25B/月**

- 5/28 SpaceX S-1 披露：Anthropic 付 xAI **每月 $12.5 亿**，独占 Colossus 1
- 220,000 颗 NVIDIA GPU，300MW
- 这是"算力外包"的天花板——大模型公司宁可付天价也要独占算力
- 算力从"可购买的商品"变成"必须签长约的资产"

**7. ARR 跳升曲线**

- Anthropic：$9B（2025 底）→ $30B（4 月）→ $47B（5 月）
- 5 个月翻 1.5 倍
- 但 OpenAI "missed revenue targets"（5 月底 CNN 报道）——具体数字未公开
- 巨头间分化：模型层内部已经是"SaaS 派 vs 咨询派"

**8. 中国出口数据的反常**

- 2026 年 4 月中国集成电路出口额 **310.85 亿美元，+100.1% YoY**
- 量只增 3.8%——**增值不增量**
- 国产芯片涨价 + 算力卡（昇腾、寒武纪）出口
- 经济学解释：中国从"芯片进口国"变成"芯片出口国"，但出口的是**中端**而非高端

**9. 谁在给谁付钱？**

- 中心-边缘的资金流：
  - 美国基金（Altimeter / Sequoia / Coatue）→ Anthropic 估值 $965B
  - Anthropic → xAI 算力 $15B/年
  - xAI → NVIDIA / TSMC / 电力公司
  - 利润最终回流到：美国私募基金 + 高管股票
- **全球南方的角色**：提供矿（钴、铜、锂）、提供数据、最后买单

**10. 利润回流的不平等**

- 一个 220,000 GPU 数据中心耗电 300MW
- 民主国家可调用算力是中国 AI 产业的 **11 倍**（Threads 引述，待核实）
- 中国试图用"垂直整合"（哈勃投资 112+ 项目）打破"中心定价"
- 这是 AI 时代的"南北问题"

### Part 2：科技层 | 幻灯片 11-16 | ~7 分钟

**11. 算力、模型、数据的"国界"**

| 层级 | 谁掌握 | 流动自由吗 |
|---|---|---|
| 算力（GPU） | NVIDIA（美）vs 华为昇腾（中） | ❌ 出口管制 |
| 基础模型 | Anthropic / OpenAI / Google（美）vs DeepSeek / Qwen / GLM（中） | ⚠️ API 限制 |
| 数据 | 各国家 + 平台 | ❌ 数据主权 |
| 应用层 | 全球开发者 | ✅ 自由流动 |

**12. 出口管制的演化**

- 2022/10：首次禁令，A100 / H100
- 2023/10：性能阈值细化
- 2025/1：AI Diffusion Rule（全球分级）
- 2025/5：Trump 政府暂缓执行
- 2026/5/31：堵漏——海外子公司也管
- **逻辑**：从"限产品"到"限实体"再到"限国家"，管制粒度越来越细

**13. 国产算力的突破时刻**

- **4/24/2026 DeepSeek V4 发布同日，华为宣布昇腾全系支持**
- 这是**国产大模型首次在国产芯片上完成训练 + 推理全栈部署**
- V4-Pro 推理时延 20ms，V4-Flash 10ms
- 字节、腾讯、阿里**加码采购**昇腾 950PR
- 黄仁勋 4/15："要是 DeepSeek 这样的成果先在华为平台上出现，对我们国家会是非常糟糕的结果。"

**14. 开源 vs 闭源的政治学**

- 中国阵营：DeepSeek / Qwen / GLM / Kimi — 全部开源权重
- 美国阵营：OpenAI / Anthropic 闭源；Meta / Mistral 半开源
- 开源不只是商业策略，是**地缘策略**：
  - 中国用开源做"市场换规则"——让全球南方用上中国模型
  - 美国用闭源做"规则换市场"——出口管制 + API 限制
- 2026 年 4 月开源权重榜：中国 GLM-5 排第一

**15. 算法即规制**

- Mythos（Anthropic 未发布模型）：Hassabis 在 Google I/O 主动承认"能让企业和政府措手不及"
- 模型本身的"价值观"成为输出
- 美国 / 欧盟的"价值观 AI" vs 中国的"发展优先 AI"
- 这不是技术分歧，是**规制权的竞争**

**16. 科技层的总结**

- 算力：美国仍主导，但中国从 5% → 45% 国产替代
- 模型：中美各有头部，中国开源领先
- 数据：国家边界 + 平台边界双重锁定
- 关键判断：**科技层不是"赢家通吃"，是"双中心 + 边缘"**

### Part 3：政治层 | 幻灯片 17-21 | ~6 分钟

**17. 三套 AI 规则**

| 维度 | 美国 | 中国 | 欧盟 |
|---|---|---|---|
| 主导逻辑 | 国家安全 | 产业自主 + 安全 | 基本权利 |
| 主要工具 | 出口管制 | 国产替代 + 数据安全 | AI 法案（风险分级） |
| 2026 关键事件 | 5/31 新规堵漏 | DeepSeek V4 国产化 | 5/7 Omnibus 推迟 |
| 对 AI 公司态度 | 友好 | 政策倾斜 | 谨慎 |

**18. 美国：小院高墙**

- 不是全面脱钩，是**精准卡脖子**
- 5/31 新规堵住海外子公司漏洞——"该管的管，不该管的继续做生意"
- 议员警告中国"通过芯片走私、模型蒸馏、窃取商业秘密"缩小差距
- 一句话总结：**美国要的是"我能卖你不能买"**

**19. 欧盟：以规则换时间**

- AI 法案是全球第一部综合性 AI 立法
- 5/7 Omnibus 协议：高风险 AI 合规从 8/2026 推到 12/2027
- 推迟原因：标准未就绪
- 一句话总结：**欧盟要的是"先立法再执行"，结果变成"先立法再延期"**

**20. 中国：垂直整合 + 内循环**

- 国产替代：从"备选项"升级为"必选项"
- 哈勃投资 112+ 项目 → 构筑国产算力供应链
- 政策倾斜：央企优先采购国产芯片
- 一句话总结：**中国要的是"我有你也有"**

**21. 三套规则的底层冲突**

- 美国的"国家安全" vs 欧盟的"基本权利" vs 中国的"产业自主"
- 没有一套能套用到其他两方
- 冲突的关键点：
  - 算力：美国不让中国买，中国不让美国卖——**双输**
  - 数据：欧盟要跨境流动保护，中国要数据主权——**难调和**
  - 模型：美国要"价值观对齐"，中国要"发展优先"——**结构性对立**

### Part 4：三个案例 | 幻灯片 22-27 | ~8 分钟

**22. 案例 1：5/31 美国新规**

- 5/31 周末 BIS 紧急发布
- 内容：凡中国总部企业的海外子公司也需许可
- 估算：过去一年数十万颗高端芯片经海外子公司流入
- 英伟达："影响有限"
- **政治经济学解读**：这是"算力地租"的国家化——美国把算力从"商品"重新定义为"战略物资"

**23. 案例 1：解读**

- 不是"全面脱钩"，是"精准卡脖子"
- 美国的目标是：**保护本国 AI 公司的领先优势**，而不是消灭中国 AI
- 中国怎么应对：从马来西亚等地"绕道"的路被堵
- **下一手**：可能转向自研 + 灰色市场

**24. 案例 2：DeepSeek V4 + 昇腾 950PR**

- 4/24 DeepSeek V4 发布，V4-Pro 1.6T 参数
- 同日华为昇腾全栈支持
- 推理时延：Pro 20ms / Flash 10ms
- 字节 / 腾讯 / 阿里加码采购
- **政治经济学解读**：这是"国产替代"从"能造"到"能用"的关键拐点

**25. 案例 2：解读**

- 不是"国产替代"的胜利，是"垂直整合"的胜利
- 哈勃投资 112 个上游项目 + 华为 30 年硬件积累 + DeepSeek 算法突破
- 缺一不可——单一突破不够，**体系协同**才有用
- 关键转折：当国产模型定价节奏（"下半年 950PR 批量上市后 Pro 价格大幅下调"）绑定国产芯片出货

**26. 案例 3：全球南方的"被定价"位置**

- 谁是边缘？拉美 / 非洲 / 中东 / 东南亚（除马来西亚）
- 他们被定价的方式：
  - API 调用费：按 token 计费，美元结算
  - 云服务费：被 AWS / Azure / 阿里云瓜分
  - 数据回流：本地用户数据被传回中心训练
- 没有任何一家全球南方公司进入 AI 模型前 20
- **政治经济学解读**：AI 时代的"原材料"（数据 + 矿 + 廉价电力）仍然被中心定价

**27. 案例 3：解读**

- 全球南方的"破局"路径：和上一轮工业化国家一样——**接受剥削直到积累资本**
- 中国 30 年走完了这条路（从"世界工厂"到"芯片出口国"）
- 印度 / 越南 / 巴西 / 沙特在走同样的路
- 但 AI 比传统工业更难追——**算力的规模效应是赢家通吃**

### 【结尾】| 幻灯片 28-30 | ~3 分钟

**28. 谁在剥削谁？——我的判断**

- **直接剥削**：中心（美/中）对边缘（全球南方）的算力 + 数据 + 利润三重抽取
- **间接剥削**：中心之间的"卡脖子"互相消耗，最终成本转嫁给全球南方
- **新型剥削**：模型本身在重塑劳动——5/30 稿讲的"异化"是这一剥削的微观层
- **剥削的形式变了**：从殖民地的直接掠夺，变成"算力地租 + 数据地租 + 平台佣金"

**29. AI 帝国主义 vs AI 公共化**

- **AI 帝国主义**：中心继续掌控算力 + 模型 + 数据，利润回流本国
- **AI 公共化**：
  - 中国方案：开源权重 + 国产替代
  - 欧盟方案：AI 法案 + 跨境数据保护
  - 联合国方案？目前没有
- 我的判断：短期 AI 帝国主义是主调，长期看中国开源是最大的"反中心"力量

**30. 三个开放问题**

1. **如果 Mythos 这类"双用途模型"出现，AI 法案 + 出口管制都拦不住——规则会失效吗？**
2. **当国产芯片定价节奏绑定大模型价格，中国的"国产替代"会不会变成新的"中心-边缘"？**
3. **全球南方什么时候会有自己的"DeepSeek"？还是永远不会有？**
