# AI学习数据化-马原课堂分享

## 主题方向

- **听众**：必修马原课同学（5/30 异化讲已讲"产品/过程/类本质/人与人"四维度，6/3 地缘讲已讲宏观结构，A 拜物教讲已归档备用，本次不重复任何一篇）
- **时长**：~15 分钟（不含互动）
- **单讲**
- **必带 2025-2026 最新案例**：复旦 AI 政策 / 知网 AIGC 3.0 / Anthropic 9.28 政策 / Duolingo Max / 周晓芸 780 元 / 学而思九章 6/1 信通院 5 级
- **必带 1 个马原原典**：《1844 年经济学哲学手稿》"活劳动是价值源泉"
- **5 个学生日常场景**（必含）：写作业 / 改语法 / 模拟面试 / 复习考试 / 写求职信
- **中国本土案例必须**（≥3 个）：复旦 / 周晓芸 / 知网 / 学而思 / 豆包
- **学者引用 3-4 个**（马克思 + Fuchs + Illich 必含）
- **互动题命中大学生体感**：80% vs 5% 反差 + "它拿走了你什么" 收尾
- 沿用 5/30 异化讲风格（"~X 分钟 | 幻灯片 X-Y" 节奏标注）

- **不重复 6/3 地缘讲**（宏观结构 / 三层地缘 / NVIDIA / Anthropic-xAI）
